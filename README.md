# Stock V2 - A股资讯分析系统

以**公司**为核心视角的A股智能分析系统。用户打开一家公司，看到它的核心定位、产业链位置、最新重要动态和AI综合研判。

## 系统定位

**用户场景：我想快速了解一家公司。**

参考 aichainmap.com 的图谱化知识库架构，围绕个股页面构建：

```
┌─────────────────────────────────────────────┐
│  公司名称：中芯国际 (688981.SH)               │
│  核心业务：晶圆代工，国内先进制程龙头            │
│  产业链位置：芯片硬件层 > 晶圆代工              │
│  上游：中微公司(设备) | 下游：紫光国微(设计)     │
│  概念：芯片 · 国产替代 · 先进制程               │
├─────────────────────────────────────────────┤
│  最新动态（只显示最重要的几条）                  │
│  [利好] Q2营收超预期，同比增长23%    06-28      │
│  [利好] 获大基金二期增资              06-20      │
│  [中性] 新厂产能爬坡中               06-15      │
├─────────────────────────────────────────────┤
│  AI研判：近期偏多，利好因素集中，关注产能释放节奏  │
└─────────────────────────────────────────────┘
```

## 核心功能

### 公司档案（核心）
- 公司核心业务描述（一句话说清楚干什么的）
- 产业链位置（五层架构中的哪一层，上下游是谁）
- 所属概念板块

### 资讯采集层（只存重要的）
- 财经新闻（东方财富、同花顺、新浪财经等）
- 上市公司公告（巨潮资讯）
- 社区讨论（雪球）
- 采集后经重要度过滤，日均5000条压缩到500~800条入库
- 不存原始全文，只存AI摘要 + 结构化结果

### 智能分析层
- 实体识别：股票代码、公司名称、行业概念
- 情感分析：利好/利空/中性
- 事件提取：业绩、并购、增减持、回购等重大事件
- 关联分析：产业链上下游传导

### AI能力层
- 个股动态摘要：最近重要资讯一句话概括
- 趋势研判：基于近期资讯的综合判断

## 技术架构

```
┌──────────────────────────────────────┐
│           前端：个股全景页             │
│      React + TypeScript + ECharts     │
├──────────────────────────────────────┤
│           API：FastAPI                │
├────────┬────────┬────────┬───────────┤
│  采集   │  分析   │  AI    │  图谱     │
│(过滤噪音)| (NLP)  |(摘要)  |(产业链)   │
├────────┴────────┴────────┴───────────┤
│      PostgreSQL + Redis + ES          │
└──────────────────────────────────────┘
```

## 快速开始

```bash
# 后端
cd backend && pip install -r requirements.txt

# 前端
cd frontend && npm install && npm run dev

# Docker一键启动
docker-compose up -d
```
