# Stock V2 - A股资讯分析系统

以**公司**为核心视角的A股智能分析系统。打开一家公司，看到它的核心定位、产业链位置、最新重要动态和AI研判。

## 产品形态

```
┌─────────────────────────────────────────────┐
│  公司名称：中芯国际 (688981.SH)               │
│  核心业务：晶圆代工，国内先进制程龙头            │
│  产业链位置：芯片硬件层 > 晶圆代工              │
│  上游：中微公司(设备) | 下游：紫光国微(设计)     │
│  概念：芯片 · 国产替代 · 先进制程               │
├─────────────────────────────────────────────┤
│  最新动态（只显示最重要的几条）                  │
│  [利好] Q2营收超预期，同比增长23%    06-28      │
│  [利好] 获大基金二期增资              06-20      │
│  [中性] 新厂产能爬坡中               06-15      │
├─────────────────────────────────────────────┤
│  AI研判：近期偏多，利好因素集中，关注产能释放节奏  │
└─────────────────────────────────────────────┘
```

## 核心功能

- **公司档案**: 核心业务、产业链位置（五层架构）、上下游、所属概念
- **重要资讯**: 经重要度过滤后的最新动态，只显示有价值的几条
- **事件时间线**: 业绩、并购、增减持等重大事件
- **AI研判**: 基于近期资讯的综合趋势判断

## 技术架构

```
前端(Vue3+Vite) → FastAPI(单进程) → 采集/分析/AI/图谱
                                       ↓
                                  SQLite(单文件)
```

## 快速开始

```bash
./app.sh start       # 启动（自动建环境、装依赖）
./app.sh stop        # 停止
./app.sh status      # 查看状态
./app.sh logs        # 查看后端日志
./app.sh restart     # 重启
```

启动后访问：
- 前端: http://localhost:35173
- API文档: http://localhost:38080/docs
