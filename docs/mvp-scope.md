# MVP分期计划

## Phase 1：单公司看板（4~6周）

**目标**: 能搜索一家公司，看到它的核心信息和最新几条重要资讯。

### 后端
- [ ] 股票基础数据导入（AKShare获取全量股票列表+行业+概念，免费无需token）
- [ ] Stock表 core_business 字段数据填充（从公司简介/年报摘要）
- [ ] 产业链基础数据导入（预定义五层架构+子行业映射）
- [ ] 东方财富采集器（新闻+快讯）
- [ ] 巨潮资讯采集器（公告）
- [ ] 重要度过滤（规则层，关键词+股票实体匹配）
- [ ] 实体识别（股票代码/名称正则匹配）
- [ ] 情感分析（规则层，利好/利空关键词）
- [ ] AI摘要生成（Claude API，每条入库资讯生成摘要）
- [ ] 个股全景接口 `/stocks/:code/profile`

### 前端
- [ ] 股票搜索页（输入代码或名称，跳转个股页）
- [ ] 个股全景页（公司档案+最新资讯+事件时间线）
- [ ] 基础布局和路由

### 基础设施
- [ ] SQLite数据库初始化脚本（自动建表）
- [ ] APScheduler定时任务配置
- [ ] 基础日志和错误处理
- [ ] `python -m app.main` 一键启动

---

## Phase 2：分析增强（3~4周）

**目标**: 分析更准确，覆盖更多数据源。

### 后端
- [ ] 雪球采集器（社区讨论）
- [ ] 事件检测（模板匹配+关键词组合）
- [ ] 情感分析升级（预训练模型层）
- [ ] 产业链上下游关联计算
- [ ] 个股关联图谱接口
- [ ] 概念热度排行接口

### 前端
- [ ] 产业链位置可视化（简版，展示上下游关系）
- [ ] 事件时间线组件
- [ ] 概念标签展示

---

## Phase 3：图谱与AI（3~4周）

**目标**: 产业链图谱可视化，AI研判能力。

### 后端
- [ ] 产业链图谱接口（五层架构完整数据）
- [ ] 概念关联图谱（基于共现关系）
- [ ] AI趋势研判（Claude API，基于近期资讯综合分析）
- [ ] 全市场概览接口

### 前端
- [ ] 产业链图谱页面（D3.js force-directed）
- [ ] AI研判展示组件
- [ ] 首页市场概览

---

## Phase 4：体验优化（持续）

- [ ] 更多数据源（同花顺、新浪财经）
- [ ] WebSocket 实时推送新资讯
- [ ] 情感分析升级到LLM层
- [ ] 搜索增强（拼音、模糊匹配）
- [ ] 数据清理定时任务
- [ ] 用户关注/收藏功能
